# Lesson 2: Variables and Data Types

In this lesson, we will explore different variable types in Python that are particularly useful in marketing data analysis and automation.

## 1. Variables
A variable in Python is used to store information that can be referenced and manipulated later. For marketing purposes, we can use variables to store data like campaign budget, email lists, or customer segments.

```python
campaign_budget = 1000  # Budget in USD
email_list = 2000  # Number of emails in a list
```

## 2. Strings
Strings are used to store text data. In marketing, we often work with names, product descriptions, and promotional messages.

```python
product_name = "Eco-Friendly Water Bottle"
description = "This water bottle is made from recycled materials and helps reduce plastic waste."
```

## 3. Numbers
In Python, numbers can be integers or floats. Numbers are essential in marketing for metrics like conversion rate, sales amounts, or instances of customer engagement.

```python
conversion_rate = 0.15  # 15% conversion rate
sales_amount = 2500.50  # Total sales in USD
```

## 4. Lists
Lists are used to store multiple items in a single variable. Lists are useful for storing customer names, campaign channels, or product features.

```python
email_campaigns = ["Spring Sale", "Summer Discount", "Fall Promotion"]
```

## 5. Dictionaries
Dictionaries are used to store key-value pairs. They are particularly useful for associating data with specific unique keys like customer data or campaign performance metrics.

```python
campaign_performance = {
    "Spring Sale": {
        "clicks": 100,
        "conversions": 15
    },
    "Summer Discount": {
        "clicks": 150,
        "conversions": 20
    }
}
```

## Conclusion
Utilizing the correct data types effectively can enhance your marketing strategies by helping you to analyze data and automate processes more efficiently.
